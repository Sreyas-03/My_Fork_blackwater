blackwater.data.circuit_to_json_graph¶

circuit_to_json_graph(circuit, available_instructions=None, node_types_to_encode=None, node_encoder=None)[source]¶

Encodes circuit to json graph data

Parameters:

  • circuit (QuantumCircuit) – circuit

  • available_instructions (Optional[List[str]]) – list of available instructions. Default all Qiskit instructions.

  • node_types_to_encode (Optional[List[type]]) – list of types of nodes to encode

  • node_encoder (Optional[NodeEncoder]) – node encoder. Default: DefaultNodeEncoder

Return type:

GraphData

Returns:

dictionary with encoded circuit as graph
